上传 ArduinoLoRa 项目到 GitHub 步骤
|-- 将 ArduinoLoRa_xxx.ino 拷贝到目录 D:\Project\SharedDevelopInfo\Arduino\ArduinoLoRaGitHub
|-- win10 搜索栏输入 git，打开 Git Bash
|-- 进入项目路径 cd d:/Project/SharedDevelopInfo/Arduino/ArduinoLoRaGitHub
|-- 添加新增项目
|   |-- git add . 
|   |-- git commit -m "写个注释" 
|   |-- git push origin master 
|   \-- git status -s 
\-- 复制新增项目的 URL，打开：https://github.com/rimelink-lorawan/ArduinoLoRaProject
